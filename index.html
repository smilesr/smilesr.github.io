This is a text-based, weather-focused game, Hot Or Not , played on your mobile phone.

A user texts the name of a city and state to the designated phone number then a text is sent out to all participants asking them to guess the current temperature in that city.

The app then queries the weather underground API, evaluates each participant's guess against the actual temperature and sends a return text informing the participant of the correctness of the guess. More features to be added.

The following tools were used for this app:

  • Ruby on Rails
  • Ruby
  • Twilio
  • Weather underground API
  • PostgreSQL
  • Heroku
