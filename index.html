This is a texting-based, weather-focused game, Hot Or Not , played on your mobile phone. A user texts the name of a city and state to a Twilio phone number then a text is sent out to all participants asking them to guess the current temperature in that city.

The app then queries the weather underground API and evaluates all answers. More features to be added. The following tools have been used so far:

  • Ruby on Rails
  • Twilio
  • Weather Underground API
  • PostgreSQL
  • Heroku
